# know your plate 

This app is designed to scan and identify microbial colonies on petri plates based on their morphology, color, and type of agar media.

## Features:
- Upload petri plate images.
- Analyze microbial colonies.
- Provides identification based on image analysis.

## How to Run:
1. Install dependencies: `pip install -r requirements.txt`
2. Start the app: `python app.py`

## Technologies Used:
- Flask
- TensorFlow (for image recognition)
- Pillow (for image processing)
